คู่มือพยาบาลตรวจการ

Nurse Supervisor Guide — โรงพยาบาลชุมชน

5 ความรู้หลัก

ที่ต้องมีในเวรตรวจการ

Module 01

การบริหารอัตรากำลัง

  • ประเมินก่อนรับเวร: IPD ผู้ป่วยหนัก 1:4 / ER ไม่เกิน 1:3
  • Patient Acuity: ประเมิน Level 1-3, ผู้ป่วย on vent
  • Float Staff: เกลี่ยคน → บันทึก Float Log
Module 02

ฉุกเฉินและอุบัติภัยหมู่

  • ประเมินขนาด: แยก 'ฉุกเฉินปกติ' กับ 'MCI'
  • ประกาศรหัส: Code Disaster → เปิด Command Post
  • START Triage: แดง ด่วนมาก / เหลือง ปานกลาง
Module 03

ระบบการส่งต่อผู้ป่วย

  • รายงาน ISBAR: Identify / Situation / Background
  • เตรียมผู้ป่วย: Stabilize V/S → เตรียมเอกสาร → Consent
  • ประสาน EMS: จัดรถพร้อม Escort
Module 04

คุณภาพและความปลอดภัย

  • จัดการทันที: เมื่อเกิดเหตุให้รักษา Situation ก่อน
  • Incident Report: บันทึก IR ภายในเวร ห้ามซ่อน/แก้
  • SIMPLE Goals: Safe Surgery, Infection Control
Module 05

กฎหมายและสิทธิผู้ป่วย

  • รับมือร้องเรียน: พาไปห้องเงียบ → ฟังไม่ขัด → ไม่โต้เถียง
  • สิทธิผู้ป่วย: รับทราบ / ปฏิเสธ (AMA) / ความลับ (PDPA)
  • จัดการศพ: ตายผิดธรรมชาติ → แจ้งตำรวจ

Checklist ก่อนส่งเวร

  • บันทึก Supervisor Log ครบถ้วน
  • รายงาน Incident Report เรียบร้อย
  • เคลียร์เคส Refer เรียบร้อย

แนวทางนิติเวชคลินิก

Clinical Forensic Medicine

Step 1

เตรียมพร้อม & สิทธิ

  • ตายผิดธรรมชาติ 5 ชนิด: ฆ่าตัวตาย, ถูกทำร้าย, สัตว์ทำร้าย, อุบัติเหตุ, ไม่ทราบสาเหตุ
  • ขอ Consent: ขอความยินยอมก่อนตรวจ
Step 2

ซักประวัติ & ตรวจร่างกาย

  • บันทึกคำพูด: บันทึก Verbatim ของผู้ป่วย ห้ามตีความ
  • ตรวจบาดแผล: บรรยายขนาด, ตำแหน่ง ลง Body Diagram
Step 3

การเก็บวัตถุพยาน

  • ถุงกระดาษ: เสื้อผ้าใส่ถุงกระดาษ ห้ามพลาสติก
  • DNA Swab: ปล่อยให้แห้งในอากาศเย็นก่อนเก็บ
Step 4

ถ่ายภาพ & เวชระเบียน

  • ถ่ายภาพ: ถ่ายก่อนล้างแผลเสมอ 3 ระยะ
  • ใช้ Scale: วางไม้บรรทัดระนาบเดียวกับแผล
Step 5

ส่งมอบหลักฐาน

  • บันทึกส่งมอบ: ใช้ Chain of Custody Form ระบุผู้ส่ง-รับ
  • ส่งชันสูตรศพ: ห้ามล้างศพ ห้ามถอดสาย IV
